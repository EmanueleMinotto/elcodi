integer compareTo( MoneyInterface $other)

Compares current Money object to another

Will return -1, 0, 1 if the amount of this Money object is respectively less than, equal to, or greater than the other.

This is useful when using it as a compare function for usort() and the likes.

static MoneyInterface create( integer $amount, CurrencyInterface $currency)

Returns a new instance of a Money object

This factory should be the only process to instantiate a object implementing MoneyInterface.

In order to return a proper Money value object, an integer amount and a Currency object implementing CurrencyInterface are needed.
