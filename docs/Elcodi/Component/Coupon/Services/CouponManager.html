CouponInterface duplicateCoupon( CouponInterface $coupon, DateTime $dateFrom = null)

Creates a new coupon instance, given an existing Coupon as reference

You can specify a DateTime new coupon will be valid from. If not specified, current DateTime will be used

If given coupon is valid forever, new coupon will also be Otherwise, this method will add to validFrom, the same interval than given Coupon

Also can be specified how new Coupon name must be defined. If none, automatic generator will add to existing name, 10 random digits.

Given Coupon name: FOO New Coupon name: FOO_a67b6786a6

Coupons are only generated, and are not persisted in Manager nor Flushed

DateTime $dateFrom Date From. If null, takes actual dateTime
