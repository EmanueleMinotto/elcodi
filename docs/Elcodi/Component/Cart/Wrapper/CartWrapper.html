class CartWrapper implements WrapperInterface

Class CartWrapper

Envelopes a Cart object and provides the minimum logic to load it from HTTP Session, from the Customer collection of pending Carts or by factoring a pristine Cart.

The most useful method in this wrapper is get(), which will behave according to different scenarios:

  • When the Customer has pending Carts, the last Cart form this collection is returned
  • When there is a Cart in session, it is associated with the Customer and is returned
  • When there is no Cart in session, a new one is factored
